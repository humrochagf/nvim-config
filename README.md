# NVim Configs

My personal nvim configs.

```shell
git clone git@github.com:humrochagf/nvim-config.git ~/.config/nvim
```

## External dependencies

Some external tools needs to be installed for all functions to work.

Most of them can be checked by running `:checkhealth`

### Clipboard

For clipboard support install `xclip`.

### Treesitter CLI

To be able to use Treesitter functions its cli tool need to be installed.

```shell
cargo install tree-sitter-cli --locked
```
